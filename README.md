# 秋璇学习总结

这是一个为秋璇建立的学习总结和记录系统，用于整理和分析学习过程中的讨论内容、问题和知识点。

## 📚 项目简介

本项目通过分析微信聊天记录，自动整理学习内容，生成学科总结和上课记录，帮助跟踪学习进度和重点问题。

## 🚀 快速开始

### 方法一：通过邮件发送聊天记录（推荐）

1. **在手机上发送聊天记录到邮件**
   - 微信 → 与秋璇的聊天 → "..." → 查找聊天记录
   - 选择日期范围（2025年9月1日至今）→ 全选 → 转发到邮件

2. **在电脑上保存邮件为EML格式**
   - 打开邮件客户端，将邮件另存为 `.eml` 文件
   - 将文件放入 `assets/chat/` 目录

3. **运行处理脚本**
   ```bash
   py scripts/parse_email_chat.py
   py scripts/parse_chat.py
   ```

详细步骤请参考：[微信聊天记录导出指导](docs/微信聊天记录导出指导.md)

## 🔧 脚本说明

### parse_chat.py
主处理脚本，用于解析聊天记录并生成学习总结文档。

**功能：**
- 解析文本、HTML格式的聊天记录
- 自动识别学科（数学、物理、化学）
- 识别提问和回答
- 生成学科总结文档
- 生成上课记录文档

**使用方法：**
```bash
py scripts/parse_chat.py
```

### parse_email_chat.py
专门处理邮件格式的聊天记录。

**功能：**
- 解析 `.eml` 邮件文件
- 提取邮件正文和附件
- 识别微信聊天记录格式
- 按日期过滤消息

**使用方法：**
```bash
py scripts/parse_email_chat.py
```

## 📝 文档说明

### 学科总结文档
每个学科文档包含：
- **时间线**：按日期组织的讨论记录
- **重点问题**：突出显示的重要问题和解答
- **知识点总结**：分类整理的知识点
- **易错点**：总结的常见错误和注意事项

### 上课记录文档
按月组织，每份记录包含：
- **日期时间**：每次"上课"的时间
- **提问内容**：您提出的问题
- **学生回答**：秋璇的回答
- **知识点**：涉及的知识点
- **教学分析**：回答质量评估、理解程度分析、需要加强的方面、后续建议

### 学习计划文档
包含：
- **当前学习进度**
- **待解决问题**
- **学习目标**
- **时间安排**
- **重点复习内容**

## 🔒 隐私保护

- `assets/chat/` 和 `assets/images/` 目录已配置为不上传到GitHub
- 所有聊天记录文件仅保存在本地
- 只有处理后的总结文档会上传到GitHub

## 📊 处理统计

- **处理时间范围**：2025年9月1日至今
- **支持格式**：文本、HTML、EML邮件文件
- **自动分类**：数学、物理、化学
- **自动识别**：提问、回答、知识点

## 🔄 更新流程

1. 导出新的聊天记录到 `assets/chat/` 目录
2. 运行处理脚本更新文档
3. 查看和编辑生成的文档
4. 提交更改到GitHub

## 📞 使用提示

- 定期更新聊天记录，保持文档最新
- 可以手动编辑生成的文档，添加更多分析
- 图片会保存在 `assets/images/` 目录，可在文档中引用
- 所有脚本都支持中文路径和文件名
