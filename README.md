# 秋璇学习总结

这是一个为秋璇建立的学习总结和记录系统，用于整理和分析学习过程中的讨论内容、问题和知识点。

## 项目结构

```
qiuxuan/
├── README.md                 # 项目说明
├── docs/                    # 学习文档目录
│   ├── 数学总结.md          # 数学学科总结
│   ├── 物理总结.md          # 物理学科总结
│   ├── 化学总结.md          # 化学学科总结
│   ├── 综合总结.md          # 综合学习总结
│   ├── 学习计划.md          # 学习计划
│   └── class_records/       # 上课记录（按月组织）
├── assets/                  # 资源文件目录
│   ├── images/             # 聊天记录中的图片
│   └── chat/               # 原始聊天记录（可选）
└── scripts/                # 处理脚本
    └── parse_chat.py       # 聊天记录解析脚本
```

## 使用说明

1. 将微信聊天记录导出后，放入 `assets/chat/` 目录
2. 运行 `scripts/parse_chat.py` 脚本处理聊天记录
3. 查看生成的文档了解学习总结和上课记录

## 文档说明

- **学科总结文档**：按学科分类整理的学习内容、问题和知识点
- **上课记录**：将提问和回答整理为教学记录，包含分析和建议
- **学习计划**：当前学习进度、目标和安排
