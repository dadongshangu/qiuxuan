## 2026-01-25（数学第25题）

### 提问内容

分段函数与直线交点问题：已知函数 `y = x² + 2x` (x > 0) 和 `y = -x` (x ≤ 0) 的图象如图所示，若直线 `y = x + m` 与该图象恰有三个交点，则 m 的取值范围为____。

![数学第25题](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/math_question_25.png)

### 学生回答

_待补充_

### 知识点

- **分段函数**：需要分段讨论交点
- **直线与分段函数的交点**：
  - x ≤ 0 部分：`y = -x` 与 `y = x + m` 的交点为 `x = -m/2`（需要 `m ≥ 0`）
  - x > 0 部分：`y = x² + 2x` 与 `y = x + m` 的交点满足 `x² + x - m = 0`
- **二次方程判别式**：`Δ = 1 + 4m`
- **三个交点的条件**：需要 x < 0 部分一个交点，x > 0 部分有合适的交点情况

### 解答要点

**关键分析**：
1. x ≤ 0 部分：当 `m > 0` 时，有一个交点 `(-m/2, m/2)`
2. x > 0 部分：方程 `x² + x - m = 0` 在 `m > 0` 时有一个正根
3. 要得到三个交点，需要 `m > 0` 且 `m < 1/4`

**答案**：`0 < m < 1/4`

## 2026-01-25（化学第14题）

### 提问内容

自制干燥剂成分分析问题：某化学项目小组探究久置后自制干燥剂的成分。自制干燥剂初始成分为CaO和NaOH。久置后可能含有CaO、Ca(OH)₂、CaCO₃、NaOH、Na₂CO₃。取样品溶于足量水，静置后，用10%的稀盐酸溶液滴定，得到CO₂质量与稀盐酸质量的关系图。下列说法错误的是？

![化学第14题](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/chemistry_question_14.png)

**选项**：
- A. 该久置后自制干燥剂样品中一定含NaOH
- B. 该久置后自制干燥剂样品中一定含Na₂CO₃和CaCO₃
- C. 图中x=18.25，且M点溶液中一定只含一种溶质
- D. 变质前自制干燥剂中CaO与NaOH的质量比为14:25

### 学生回答

_待补充_

### 知识点

- **干燥剂变质反应**：
  - CaO + H₂O = Ca(OH)₂（生石灰吸水）
  - Ca(OH)₂ + CO₂ = CaCO₃ + H₂O（氢氧化钙与CO₂反应）
  - 2NaOH + CO₂ = Na₂CO₃ + H₂O（氢氧化钠与CO₂反应）

- **碳酸盐与盐酸反应**：
  - Na₂CO₃ + HCl = NaHCO₃ + NaCl（第一步，不产生CO₂）
  - NaHCO₃ + HCl = NaCl + CO₂↑ + H₂O（第二步，产生CO₂）
  - CaCO₃ + 2HCl = CaCl₂ + CO₂↑ + H₂O（直接产生CO₂）

- **图表分析**：
  - 0-14.60g HCl阶段：无CO₂产生（中和反应或Na₂CO₃第一步）
  - 14.60g开始：产生CO₂（NaHCO₃和CaCO₃与HCl反应）
  - M点：反应过程中，溶液中含多种溶质

### 解答要点

**关键理解**：
1. Na₂CO₃与HCl反应分两步，第一步不产生CO₂
2. 14.60g HCl中必须有一部分用于Na₂CO₃的第一步反应，说明样品一定含Na₂CO₃
3. M点处于反应过程中，溶液中不可能只含一种溶质

**答案**：**C**

**原因**：M点溶液中必然含有多种溶质（如NaCl、CaCl₂、NaHCO₃等），不可能只含一种溶质。

## 2026-01-25（物理第4题：自动门电路设计）

### 提问内容

自动门电路设计问题：应用电学知识设计学校门口的"自动门"，需要自动进行健康码和体温验证。

**可用器材**：
- 电源
- 手动开关 S
- 开关 S₁：感应到体温正常时可闭合
- 开关 S₂：感应到健康码为黄色或红色时可闭合
- 开关 S₃：感应到健康码为绿色时可闭合
- 红灯灯泡 L
- 电动机 M：工作时驱动门打开

请画出符合上述要求的电路图。

### 学生回答

_待补充_

### 知识点

- **串联电路**：实现"与"逻辑（AND逻辑）
  - S₁和S₃串联：只有体温正常**且**健康码为绿色时，M才工作
- **并联电路**：实现独立控制
  - 开门电路和警告电路并联：互不影响
- **电路设计步骤**：
  1. 分析功能要求
  2. 确定逻辑关系（"且"→串联，"或"→并联）
  3. 选择连接方式
  4. 绘制电路图

### 解答要点

**关键理解**：
1. **开门条件**：体温正常（S₁闭合）**且**健康码为绿色（S₃闭合）
   - 这需要S₁和S₃**串联**，只有两者都闭合时，M才工作
2. **警告功能**：健康码为黄色或红色（S₂闭合）时，红灯L亮起
   - S₂与L串联，独立控制
3. **总开关**：手动开关S控制整个电路

**电路结构**：
```
电源正极 → 手动开关S → 分两路：
  ├─ 一路：S₁ 和 S₃ 串联 → 电动机M → 电源负极
  └─ 另一路：S₂ → 红灯L → 电源负极
```

**答案要点**：
- 总开关S串联在电源正极之后
- S₁和S₃串联后与M串联（开门电路）
- S₂与L串联（警告电路）
- 开门电路和警告电路并联

## 2026-01-25（数学第22题：抛物线综合题）

### 提问内容

抛物线综合题：如图，抛物线 `y = ax² + bx + c` 经过四边形ABCD的顶点A(0,3)、B(-1,0)、D(2,3)，与x轴交于另一点E（除了B）。直线l过点E，将四边形ABCD分成面积相等的两部分，与抛物线交于另一点F。点P是抛物线上在直线l上方的动点，横坐标为t。

![数学第22题](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/math_question_22.png)

**问题**：
1. 求抛物线的表达式
2. 当t为何值时，△PFE的面积最大？并求这个最大面积的立方根
3. 是否存在点P使得△PAE为直角三角形？如果存在，直接写出t的值；如果不存在，说明理由

### 学生回答

_待补充_

### 知识点

- **抛物线解析式的确定**：利用已知点建立方程组
- **面积平分线**：过顶点的直线平分面积，需要经过对边的中点
- **三角形面积计算**：使用坐标公式或向量叉积
- **函数最值**：通过求导找极值点
- **直角三角形判定**：利用向量垂直（点积为0）

### 解答要点

**第(1)问：求抛物线表达式**
- 利用A(0,3)、B(-1,0)、D(2,3)建立方程组
- 解得：`a = -1`，`b = 2`，`c = 3`
- 抛物线表达式：`y = -x² + 2x + 3`

**第(2)问：求△PFE面积最大值**
- 确定四边形ABCD的顶点，计算面积
- 确定直线l的方程（过E且平分面积）
- 求直线l与抛物线的交点F
- 建立面积函数S(t)，其中点P(t, -t² + 2t + 3)在直线l上方
- 通过求导找最大值点：`t = 13/6`
- 最大面积的立方根：`5/6`

**第(3)问：判断△PAE是否为直角三角形**
- 讨论三种可能的直角情况：∠PAE、∠APE、∠AEP
- 利用向量点积为0判断垂直
- 情况1（∠PAE = 90°）：`t = 1` 满足条件
- 情况2（∠APE = 90°）：需要进一步计算
- 情况3（∠AEP = 90°）：`t = -2` 不满足定义域

**答案**：
1. `y = -x² + 2x + 3`
2. 当 `t = 13/6` 时面积最大，最大面积的立方根为 `5/6`

## 2026-01-25（菱形几何题）

### 提问内容

菱形几何题：已知菱形ABCD中，AB=8，点G是对角线BD上一点，CG交BA的延长线于点F，CG与CD交于点E。

![菱形几何题](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/math_rhombus_question.png)

**问题**：
1. 求证：CG² = GE · GF
2. 如果DG = 1/2 GB，且AG ⊥ BF，求cosF

### 学生回答

_待补充_

### 知识点

- **菱形的性质**：对边平行，邻边相等
- **平行线的性质**：内错角相等
- **相似三角形的判定**：两角对应相等
- **相似三角形的性质**：对应边成比例
- **割线定理**：通过相似三角形证明CG² = GE · GF

### 解答要点

**第(1)问：证明CG² = GE · GF**

**关键步骤**：
1. 利用菱形性质：CD // AB
2. 利用平行线性质：∠DCE = ∠F（内错角），∠ECG = ∠GFC
3. 证明△CGE ∽ △FGC：
   - ∠CGE = ∠FGC（对顶角相等）
   - ∠ECG = ∠GFC（已证）
4. 由相似得：CG/GF = GE/CG
5. 即：CG² = GE · GF ✓

**第(2)问：在给定条件下求cosF**

**已知条件**：
- DG = 1/2 GB（即BG : GD = 2 : 1）
- AG ⊥ BF
- AB = 8

**解题思路**：
1. 利用第(1)问的相似三角形比例关系
2. 利用AG ⊥ BF的垂直关系
3. 在相关三角形中建立方程
4. 最终求出cosF

**答案**：需要根据具体计算确定（通常为 `cosF = √3/3` 或 `cosF = 1/2`）
