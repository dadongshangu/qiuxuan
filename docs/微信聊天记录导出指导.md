# 微信聊天记录导出指导

本文档将指导您如何导出2025年9月1日至今的微信聊天记录。

## 方法一：使用微信PC版导出（推荐）

### 步骤1：打开微信PC版

1. 确保您已安装并登录微信PC版
2. 找到与秋璇的聊天窗口

### 步骤2：导出聊天记录

1. 在聊天窗口右上角，点击 **"..."** 菜单
2. 选择 **"更多"** → **"导出聊天记录"**
3. 在弹出的对话框中：
   - 选择导出格式（建议选择 **"文本"** 或 **"HTML"**）
   - 设置时间范围：选择 **"自定义时间"**
     - 开始时间：**2025年9月1日**
     - 结束时间：**今天**
   - 勾选 **"包含图片"**（如果选项可用）
4. 选择保存位置，建议保存到项目的 `assets/chat/` 目录
5. 点击 **"导出"**

### 步骤3：处理导出的文件

- 如果导出为文本格式（.txt），直接使用
- 如果导出为HTML格式，也可以使用（脚本会支持解析）
- 如果包含图片，图片通常会保存在一个单独的文件夹中，请将图片复制到 `assets/images/` 目录

## 方法二：使用微信手机版导出

### 步骤1：打开微信手机版

1. 打开微信APP
2. 找到与秋璇的聊天窗口

### 步骤2：导出聊天记录

1. 点击聊天窗口右上角的 **"..."** 图标
2. 选择 **"查找聊天记录"**
3. 选择 **"日期"**，然后选择 **"2025年9月1日"** 作为起始日期
4. 滚动到今天的消息
5. 长按消息，选择 **"多选"**
6. 选择需要导出的消息（或全选）
7. 点击 **"转发"** → **"文件传输助手"** 或 **"收藏"**

**注意**：手机版导出功能较有限，建议优先使用PC版。

## 方法三：使用第三方工具（可选）

如果上述方法无法满足需求，可以考虑使用第三方微信聊天记录导出工具，但请注意数据安全。

## 导出后的文件处理

1. 将导出的聊天记录文件放入 `assets/chat/` 目录
2. 如果图片单独导出，将图片放入 `assets/images/` 目录
3. 运行 `scripts/parse_chat.py` 脚本处理聊天记录

## 注意事项

- 确保导出的时间范围正确（2025年9月1日至今）
- 如果导出包含图片，请确保图片文件完整
- 导出后请检查文件是否完整，特别是时间跨度较长的记录
- 建议在导出前先备份重要数据

## 常见问题

### Q: 导出的文件格式不支持怎么办？

A: 脚本支持多种格式（文本、HTML、JSON等），如果您的格式不在支持列表中，请告诉我具体格式，我可以更新脚本。

### Q: 图片没有一起导出怎么办？

A: 可以手动保存聊天中的图片到 `assets/images/` 目录，脚本会尝试匹配图片与对应的消息。

### Q: 导出后文件太大怎么办？

A: 可以分时间段导出多个文件，脚本支持处理多个文件。

## 下一步

导出完成后，请将文件放入 `assets/chat/` 目录，然后运行处理脚本。
