## 重点问题

### 问题1：三孔插座、开关和指示灯的连接（2026-01-11）

**题目**：如图甲是一个带开关和指示灯（连有限流电阻的发光二极管）的三孔插座。小明发现：
1. 当开关断开时，指示灯不发光，插孔也不能提供工作电压
2. 当开关闭合时，指示灯发光，插孔可以提供工作电压
3. 如果指示灯损坏，开关闭合时插孔也能提供工作电压

请画出插座中的开关、指示灯和插孔的连接方式。

![第13题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_13.png)

**解答思路**：

根据题目条件分析：

1. **条件1分析**：开关断开时，指示灯不发光，插孔不能提供工作电压
   - 说明开关必须串联在电路中，控制指示灯和插孔

2. **条件2分析**：开关闭合时，指示灯发光，插孔可以提供工作电压
   - 说明开关闭合后，指示灯和插孔都能正常工作

3. **条件3分析**：指示灯损坏时，开关闭合时插孔仍能提供工作电压
   - 这是关键条件！说明指示灯和插孔是**并联**关系
   - 如果它们是串联，指示灯损坏（断路），整个电路断开，插孔无法工作
   - 只有并联时，指示灯损坏不影响插孔工作

**连接方式**：

```
火线 ──→ 开关 ──┬──→ 指示灯 ──→ 零线
                 │
                 └──→ 插孔（火线端）──→ 零线
                 
插孔（零线端）──→ 零线（直接连接）
插孔（地线端）──→ 地线（直接连接）
```

**具体连接步骤**：

1. **开关连接**：
   - 开关一端接火线
   - 开关另一端作为控制端

2. **指示灯连接**：
   - 指示灯一端接开关的控制端（火线侧）
   - 指示灯另一端接零线

3. **插孔连接**：
   - 插孔的火线端接开关的控制端（与指示灯并联）
   - 插孔的零线端直接接零线
   - 插孔的地线端直接接地线

**关键知识点**：
- **串联电路**：用电器依次连接，一个损坏会影响整个电路
- **并联电路**：用电器并列连接，一个损坏不影响其他用电器
- **三孔插座**：火线（L）、零线（N）、地线（E）
- **开关位置**：通常串联在火线上，控制整个电路

**易错点**：
- 误认为指示灯和插孔串联（不符合条件3）
- 忘记插孔的零线端和地线端需要直接连接
- 开关位置放错（应该控制火线）

**学生解答**：

根据题目要求，实际连接方式如下（见上图红色连接线）：

![第13题答案图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_13_answer.png)

**连接方式**：

1. **火线连接**：
   - 火线 → 开关的一个端子

2. **开关输出端连接**（开关的另一个端子分两路）：
   - 一路 → 指示灯的输入端（限流电阻的一端）
   - 一路 → 三孔插座的火线端（L）

3. **指示灯连接**：
   - 指示灯输入端（限流电阻）→ LED阳极 → LED阴极 → 零线

4. **三孔插座连接**：
   - 火线端（L）→ 接开关输出端（与指示灯并联）
   - 零线端（N）→ 直接接零线
   - 地线端（E）→ 直接接地线

**连接验证**：
- ✅ 开关断开时：指示灯和插孔都不工作（符合条件1）
- ✅ 开关闭合时：指示灯发光，插孔提供电压（符合条件2）
- ✅ 指示灯损坏时：插孔仍能提供电压（符合条件3，证明是并联）

**关键理解**：
- 开关控制火线，串联在电路中
- 指示灯和插孔的火线端并联在开关输出端
- 这样设计的好处：指示灯损坏不影响插孔使用

**图示说明**：
- 红色线条表示实际连接方式
- 开关控制整个电路的通断
- 指示灯和插孔的火线端并联，实现独立工作

### 问题2：磁悬浮地球仪（2026-01-11）

**题目**：如图甲所示是一个磁悬浮地球仪，球体内有一个条形磁体，其磁场方向与地磁场方向相同，下方环形底座内有一个电磁铁，通过磁极间的相互作用使地球仪悬浮在空中，图乙是其内部结构示意图。请在图乙中标出：
1. 球体内条形磁体的N极
2. 开关闭合后，电磁铁的磁感线方向
3. 电源的"+"、"-"极

![第14题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_14.png)

**解答思路**：

#### 第一步：确定条形磁体的N极位置

**关键信息**：球体内条形磁体的磁场方向与地磁场方向相同

**"磁场方向与地磁场方向相同"的含义**：

1. **地磁场的方向**：
   - 地磁场方向：从地理南极指向地理北极
   - 磁感线从地磁S极（地理南极附近）指向地磁N极（地理北极附近）
   - 在地球仪上：地理北极在上方，地理南极在下方

2. **条形磁体的磁场方向**：
   - 条形磁体的磁感线从N极出发，回到S极
   - 在磁体外部：从N极指向S极
   - 在磁体内部：从S极指向N极

3. **"方向相同"的含义**：
   - 条形磁体的磁感线方向与地磁场的磁感线方向一致
   - 条形磁体的N极指向地理北极方向（上方）
   - 条形磁体的S极指向地理南极方向（下方）

4. **具体理解**：
   - 在地球仪中，地理北极在上方 → 条形磁体的N极在上方
   - 地理南极在下方 → 条形磁体的S极在下方
   - 这样条形磁体的磁场方向就与地磁场方向一致

5. **为什么这样设计**：
   - 模拟真实地球的磁场
   - 使地球仪更像真实地球
   - 便于理解地磁场概念

**结论**：
- 地磁场方向：从地理南极指向地理北极（磁感线从S极指向N极）
- 地球仪上，地理北极在上方，地理南极在下方
- 因此，条形磁体的N极应该在**上方**（对应地理北极），S极在下方

**答案1**：条形磁体的N极在球体**上方**

#### 第二步：确定电磁铁的磁极

**悬浮原理**：地球仪能够悬浮，说明电磁铁对条形磁体产生**排斥力**

- 条形磁体的S极在下方（靠近电磁铁）
- 要产生排斥力，电磁铁的上端（靠近地球仪的一端）必须是**S极**
- 因此，电磁铁的下端是**N极**

#### 第三步：确定磁感线方向

**磁感线规律**：
- 磁感线从N极出发，回到S极
- 在磁体外部：从N极指向S极
- 在磁体内部：从S极指向N极

**电磁铁的磁感线方向**：
- 电磁铁上端是S极，下端是N极
- 磁感线从下端的N极出发，向上指向S极
- 在电磁铁内部，磁感线方向：**从S极（上端）指向N极（下端）**

**答案2**：电磁铁的磁感线方向在内部是**从S极指向N极**（从上指向下）

#### 第四步：确定电源正负极（使用右手螺旋定则）

**右手螺旋定则**：
- 用右手握住螺线管
- 四指指向电流方向
- 拇指指向N极（磁场方向）

**分析过程**：
1. 已知电磁铁下端是N极，上端是S极
2. 根据右手螺旋定则，拇指指向N极（下端）
3. 四指指向电流方向（在螺线管中）
4. 观察螺线管的绕线方向，确定电流流向
5. 根据电流方向，确定电源正负极

**具体判断**：
- 如果电流从电源正极流出，经过开关，进入螺线管
- 根据绕线方向和右手螺旋定则，确定磁场方向
- 调整电源正负极，使磁场方向符合要求（下端N极，上端S极）

**答案3**：电源正极（+）连接开关的一端，电源负极（-）连接螺线管的另一端

**关键知识点**：
- **磁悬浮原理**：同名磁极相互排斥
- **地磁场方向**：从地理南极指向地理北极
- **磁感线方向**：外部从N极到S极，内部从S极到N极
- **右手螺旋定则**：判断通电螺线管的磁场方向
- **磁极判断**：根据悬浮条件（排斥力）确定磁极

**易错点**：
- 混淆地磁场方向和条形磁体方向
- 磁感线方向判断错误（内部和外部方向相反）
- 右手螺旋定则应用错误
- 电源正负极判断错误

**学生解答**：

**标注结果**：

1. **条形磁体的N极**：
   - 位置：球体**上方**（对应地理北极）
   - 原因：磁场方向与地磁场相同

2. **电磁铁的磁感线方向**：
   - 方向：在电磁铁内部，从**S极（上端）指向N极（下端）**
   - 外部：从N极（下端）出发，向上指向S极（上端）

3. **电源正负极**：
   - 根据右手螺旋定则和绕线方向确定
   - 电源正极（+）：连接开关的一端
   - 电源负极（-）：连接螺线管的另一端

**解题步骤总结**：
1. 根据地磁场方向确定条形磁体N极位置
2. 根据悬浮条件（排斥力）确定电磁铁磁极
3. 根据磁感线规律确定磁感线方向
4. 根据右手螺旋定则确定电源正负极

**关键理解**：
- 磁悬浮需要同名磁极相互排斥
- 地磁场方向是判断条形磁体方向的关键
- 右手螺旋定则是连接电流和磁场的桥梁

## 知识点总结

### 家庭电路

#### 三孔插座
- **火线（L）**：提供工作电压，有电危险
- **零线（N）**：与火线形成回路，正常情况下不带电
- **地线（E）**：保护接地，防止漏电

#### 开关连接
- 开关必须串联在火线上
- 控制整个电路的通断
- 断开时，用电器不带电，更安全

#### 用电器连接方式
- **串联**：一个损坏，整个电路断开
- **并联**：一个损坏，不影响其他用电器正常工作

### 电磁铁

#### 基本结构
- 电源、开关、螺线管（线圈）串联

#### 工作原理
- 通电螺线管产生磁场
- 磁场方向由电流方向决定

#### 右手螺旋定则
- 用右手握住螺线管
- 四指指向电流方向
- 拇指指向N极（磁场方向）

### 磁悬浮

#### 基本原理
- **同名磁极相互排斥**：N-N或S-S之间产生排斥力
- **异名磁极相互吸引**：N-S之间产生吸引力
- 磁悬浮利用**排斥力**使物体悬浮

#### 地磁场
- 地磁场方向：从地理南极指向地理北极
- 地理北极对应地磁场的N极方向
- 地理南极对应地磁场的S极方向

**"磁场方向相同"的理解**：
- 当条形磁体的磁场方向与地磁场方向相同时，意味着：
  - 条形磁体的N极指向地理北极方向（上方）
  - 条形磁体的S极指向地理南极方向（下方）
  - 条形磁体的磁感线方向与地磁场的磁感线方向一致
- 这样设计是为了模拟真实地球的磁场，使地球仪更像真实地球
- **记忆方法**：
  - 地磁场：从地理南极指向地理北极
  - 条形磁体：N极在上（对应地理北极），S极在下（对应地理南极）
  - 两者方向一致
- **易混淆点**：
  - 地理北极 ≠ 地磁N极（地磁N极在地理南极附近）
  - 但题目中说的是"磁场方向相同"，指的是磁感线的方向一致
  - 在地球仪上，条形磁体的N极在上方，对应地理北极方向

#### 磁感线方向
- **磁体外部**：从N极出发，回到S极
- **磁体内部**：从S极指向N极
- 磁感线是闭合曲线，形成回路

#### 磁悬浮地球仪
- 球体内条形磁体：N极在上（对应地理北极），S极在下
- 底座电磁铁：上端S极，下端N极（与条形磁体S极排斥）
- 通过控制电流方向，调节电磁铁磁极，实现悬浮

### 作图技巧

1. **电路图规范**：
   - 使用标准电路符号
   - 导线横平竖直，避免交叉
   - 连接点要清晰标注

2. **标注要求**：
   - 电源正负极要明确
   - 电流方向用箭头标注
   - 关键元件要标注名称

3. **分析步骤**：
   - 先分析题目条件
   - 确定连接方式（串联/并联）
   - 再画电路图
   - 最后检查是否符合所有条件
