### 问题1：三孔插座、开关和指示灯的连接（2026-01-25）

**题目**：如图甲是一个带开关和指示灯（连有限流电阻的发光二极管）的三孔插座。小明发现：
1. 当开关断开时，指示灯不发光，插孔也不能提供工作电压
2. 当开关闭合时，指示灯发光，插孔可以提供工作电压
3. 如果指示灯损坏，开关闭合时插孔也能提供工作电压

请画出插座中的开关、指示灯和插孔的连接方式。

![第13题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_13.png)

**解答思路**：

根据题目条件分析：

1. **条件1分析**：开关断开时，指示灯不发光，插孔不能提供工作电压
   - 说明开关必须串联在电路中，控制指示灯和插孔

2. **条件2分析**：开关闭合时，指示灯发光，插孔可以提供工作电压
   - 说明开关闭合后，指示灯和插孔都能正常工作

3. **条件3分析**：指示灯损坏时，开关闭合时插孔仍能提供工作电压
   - 这是关键条件！说明指示灯和插孔是**并联**关系
   - 如果它们是串联，指示灯损坏（断路），整个电路断开，插孔无法工作
   - 只有并联时，指示灯损坏不影响插孔工作

**连接方式**：
- 开关串联在火线上
- 指示灯和插孔并联（指示灯损坏不影响插孔工作）
- 三孔插座：火线（L）、零线（N）、地线（E）

**知识点**：
- **串联电路**：电流路径唯一，一个元件损坏，整个电路断开
- **并联电路**：电流路径多条，一个元件损坏，其他元件仍可工作
- **家庭电路连接**：
  - 开关串联在火线上（安全考虑）
  - 用电器并联（互不影响）
  - 三孔插座：左零右火上地

**易错点**：
- 误认为指示灯和插孔串联（不符合条件3）
- 忘记开关必须串联在火线上
- 三孔插座的接线顺序：左零右火上地

**学习建议**：
1. 理解串联和并联的区别
2. 掌握家庭电路的基本连接方式
3. 注意题目中的关键条件（特别是条件3）
4. 多练习电路图绘制

### 问题2：磁悬浮地球仪的磁极和电路标注（2026-01-25）

**题目**：磁悬浮地球仪中，球体内有一个条形磁体，其磁场方向与地磁场方向相同。请：
1. 标出球体内条形磁体的N极
2. 标出开关闭合后电磁铁的磁感线方向
3. 标出电源的正负极

![第14题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_14.png)

**解答思路**：

1. **条形磁体的N极判断**：
   - 题目说明：球体内条形磁体的磁场方向与地磁场方向相同
   - 地磁场方向：从地理南极指向地理北极（即地磁N极在地理南极附近）
   - 因此，条形磁体的N极应该指向地理北极（即向上）

2. **磁悬浮原理**：
   - 磁悬浮利用同名磁极相互排斥的原理
   - 球体内条形磁体的N极向上，则电磁铁的N极也应该向上（排斥）
   - 根据磁感线方向：外部从N到S，内部从S到N
   - 电磁铁的磁感线方向：外部从N极（上）到S极（下），内部从S极（下）到N极（上）

3. **电源正负极判断**：
   - 使用右手螺旋定则（安培定则）
   - 右手握住螺线管，四指指向电流方向，拇指指向N极
   - 如果N极在上，拇指向上，四指（电流方向）从下往上绕
   - 因此，电源正极在上，负极在下

**知识点**：
- **地磁场方向**：从地理南极指向地理北极（地磁N极在地理南极附近）
- **磁悬浮原理**：同名磁极相互排斥
- **磁感线方向**：
  - 外部：从N极到S极
  - 内部：从S极到N极（形成闭合回路）
- **右手螺旋定则**（安培定则）：
  - 右手握住螺线管，四指指向电流方向，拇指指向N极
  - 用于判断通电螺线管的磁极方向

**易错点**：
- 混淆地磁场方向（地磁N极在地理南极附近）
- "磁场方向与地磁场方向相同"的理解：指的是条形磁体的N极指向与地磁场方向一致
- 磁感线方向判断错误（外部从N到S，内部从S到N）
- 右手螺旋定则的应用错误

**学习建议**：
1. 理解地磁场的基本知识
2. 掌握磁悬浮的工作原理
3. 熟练运用右手螺旋定则
4. 理解磁感线的方向和闭合性

### 问题3：自动门电路设计（2026-01-25）

**题目**：应用电学知识设计学校门口的"自动门"，需要自动进行健康码和体温验证。

**可用器材**：
- 电源
- 手动开关 S
- 开关 S₁：感应到体温正常时可闭合
- 开关 S₂：感应到健康码为黄色或红色时可闭合
- 开关 S₃：感应到健康码为绿色时可闭合
- 红灯灯泡 L
- 电动机 M：工作时驱动门打开

请画出符合上述要求的电路图。

**解答思路**：

#### 第一步：分析功能要求

根据题目描述，需要实现以下功能：

1. **开门条件**：当体温正常（S₁闭合）**且**健康码为绿色（S₃闭合）时，电动机M工作，门打开
   - 这需要S₁和S₃**串联**，只有两者都闭合时，M才工作

2. **警告功能**：当健康码为黄色或红色（S₂闭合）时，红灯L亮起，表示警告
   - S₂闭合时，L亮起，这是**独立**的控制电路

3. **总开关**：手动开关S控制整个电路的总开关

#### 第二步：设计电路结构

**电路设计**：

```
电源正极 → 手动开关S → 分两路：
  ├─ 一路：S₁ 和 S₃ 串联 → 电动机M → 电源负极
  └─ 另一路：S₂ → 红灯L → 电源负极
```

**电路逻辑**：
- **开门电路**：电源 → S → S₁ → S₃ → M → 电源负极
  - 只有当S、S₁、S₃都闭合时，M才工作
  - 这确保了体温正常且健康码为绿色时，门才会打开

- **警告电路**：电源 → S → S₂ → L → 电源负极
  - 当S和S₂都闭合时，L亮起
  - 这表示健康码为黄色或红色时的警告

#### 第三步：绘制电路图

**电路图要点**：

1. **总开关S**：串联在电源正极之后，控制整个电路
2. **开门电路**：S₁和S₃串联后与M串联
3. **警告电路**：S₂与L串联
4. **两路并联**：开门电路和警告电路并联在S之后

**电路符号**：
- 电源：用电池符号或电源符号表示
- 开关：用单刀单掷开关符号
- 电动机：用M表示，带圆圈
- 灯泡：用L表示，带圆圈

**知识点**：
- **串联电路**：电流路径唯一，所有元件依次连接
  - S₁和S₃串联：只有两者都闭合，电流才能通过
  - 应用：实现"与"逻辑（AND逻辑）
  
- **并联电路**：电流路径多条，各元件并列连接
  - 开门电路和警告电路并联：互不影响，可以同时工作
  - 应用：实现独立控制

- **开关的作用**：
  - 总开关S：控制整个电路的通断
  - 传感器开关（S₁、S₂、S₃）：根据检测结果自动闭合

**易错点**：
- 误将S₁和S₃并联（这样只要有一个闭合，M就工作，不符合"且"的要求）
- 忘记总开关S（电路设计不完整）
- 将开门电路和警告电路串联（这样两者会相互影响）
- 混淆"与"逻辑和"或"逻辑：
  - "且"（AND）→ 串联
  - "或"（OR）→ 并联

**学习建议**：
1. 理解串联和并联的逻辑关系：
   - 串联 = "且"（AND）：所有条件都满足
   - 并联 = "或"（OR）：任一条件满足
2. 掌握电路设计的基本步骤：
   - 分析功能要求
   - 确定逻辑关系
   - 选择连接方式（串联/并联）
   - 绘制电路图
3. 注意题目中的关键词：
   - "且"、"和" → 串联
   - "或" → 并联
4. 多练习电路设计题，熟悉常见应用场景

## 知识点总结

### 电路连接方式

#### 串联电路
- **特点**：电流路径唯一，各元件依次连接
- **性质**：一个元件损坏，整个电路断开
- **应用**：
  - 开关串联在火线上（控制电路）
  - 实现"与"逻辑（AND逻辑）：所有条件都满足才工作

#### 并联电路
- **特点**：电流路径多条，各元件并列连接
- **性质**：一个元件损坏，其他元件仍可工作
- **应用**：
  - 家庭电路中的用电器并联（互不影响）
  - 实现"或"逻辑（OR逻辑）：任一条件满足就工作

#### 逻辑关系与电路连接
- **"且"（AND）逻辑**：需要所有条件都满足 → 使用**串联**
  - 例如：体温正常**且**健康码为绿色 → S₁和S₃串联
- **"或"（OR）逻辑**：任一条件满足即可 → 使用**并联**
  - 例如：健康码为黄色**或**红色 → S₂控制警告灯

### 家庭电路
- **三孔插座**：左零右火上地（L-N-E）
- **开关位置**：串联在火线上（安全考虑）
- **用电器连接**：并联（互不影响）

### 磁场和磁感线
- **地磁场方向**：从地理南极指向地理北极
- **磁感线方向**：
  - 外部：从N极到S极
  - 内部：从S极到N极（形成闭合回路）
- **磁悬浮原理**：同名磁极相互排斥

### 右手螺旋定则（安培定则）
- **用途**：判断通电螺线管的磁极方向
- **方法**：右手握住螺线管，四指指向电流方向，拇指指向N极
