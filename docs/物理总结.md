## 重点问题

### 问题1：三孔插座、开关和指示灯的连接（2026-01-11）

**题目**：如图甲是一个带开关和指示灯（连有限流电阻的发光二极管）的三孔插座。小明发现：
1. 当开关断开时，指示灯不发光，插孔也不能提供工作电压
2. 当开关闭合时，指示灯发光，插孔可以提供工作电压
3. 如果指示灯损坏，开关闭合时插孔也能提供工作电压

请画出插座中的开关、指示灯和插孔的连接方式。

![第13题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_13.png)

**解答思路**：

根据题目条件分析：

1. **条件1分析**：开关断开时，指示灯不发光，插孔不能提供工作电压
   - 说明开关必须串联在电路中，控制指示灯和插孔

2. **条件2分析**：开关闭合时，指示灯发光，插孔可以提供工作电压
   - 说明开关闭合后，指示灯和插孔都能正常工作

3. **条件3分析**：指示灯损坏时，开关闭合时插孔仍能提供工作电压
   - 这是关键条件！说明指示灯和插孔是**并联**关系
   - 如果它们是串联，指示灯损坏（断路），整个电路断开，插孔无法工作
   - 只有并联时，指示灯损坏不影响插孔工作

**连接方式**：
- 开关串联在火线上
- 指示灯和插孔并联（指示灯损坏不影响插孔工作）
- 三孔插座：火线（L）、零线（N）、地线（E）

**知识点**：
- **串联电路**：电流路径唯一，一个元件损坏，整个电路断开
- **并联电路**：电流路径多条，一个元件损坏，其他元件仍可工作
- **家庭电路连接**：
  - 开关串联在火线上（安全考虑）
  - 用电器并联（互不影响）
  - 三孔插座：左零右火上地

**易错点**：
- 误认为指示灯和插孔串联（不符合条件3）
- 忘记开关必须串联在火线上
- 三孔插座的接线顺序：左零右火上地

**学习建议**：
1. 理解串联和并联的区别
2. 掌握家庭电路的基本连接方式
3. 注意题目中的关键条件（特别是条件3）
4. 多练习电路图绘制

### 问题2：磁悬浮地球仪的磁极和电路标注（2026-01-11）

**题目**：磁悬浮地球仪中，球体内有一个条形磁体，其磁场方向与地磁场方向相同。请：
1. 标出球体内条形磁体的N极
2. 标出开关闭合后电磁铁的磁感线方向
3. 标出电源的正负极

![第14题图](https://cdn.jsdelivr.net/gh/dadongshangu/CDN@master/qiuxuan/physics_question_14.png)

**解答思路**：

1. **条形磁体的N极判断**：
   - 题目说明：球体内条形磁体的磁场方向与地磁场方向相同
   - 地磁场方向：从地理南极指向地理北极（即地磁N极在地理南极附近）
   - 因此，条形磁体的N极应该指向地理北极（即向上）

2. **磁悬浮原理**：
   - 磁悬浮利用同名磁极相互排斥的原理
   - 球体内条形磁体的N极向上，则电磁铁的N极也应该向上（排斥）
   - 根据磁感线方向：外部从N到S，内部从S到N
   - 电磁铁的磁感线方向：外部从N极（上）到S极（下），内部从S极（下）到N极（上）

3. **电源正负极判断**：
   - 使用右手螺旋定则（安培定则）
   - 右手握住螺线管，四指指向电流方向，拇指指向N极
   - 如果N极在上，拇指向上，四指（电流方向）从下往上绕
   - 因此，电源正极在上，负极在下

**知识点**：
- **地磁场方向**：从地理南极指向地理北极（地磁N极在地理南极附近）
- **磁悬浮原理**：同名磁极相互排斥
- **磁感线方向**：
  - 外部：从N极到S极
  - 内部：从S极到N极（形成闭合回路）
- **右手螺旋定则**（安培定则）：
  - 右手握住螺线管，四指指向电流方向，拇指指向N极
  - 用于判断通电螺线管的磁极方向

**易错点**：
- 混淆地磁场方向（地磁N极在地理南极附近）
- "磁场方向与地磁场方向相同"的理解：指的是条形磁体的N极指向与地磁场方向一致
- 磁感线方向判断错误（外部从N到S，内部从S到N）
- 右手螺旋定则的应用错误

**学习建议**：
1. 理解地磁场的基本知识
2. 掌握磁悬浮的工作原理
3. 熟练运用右手螺旋定则
4. 理解磁感线的方向和闭合性

## 知识点总结

### 电路连接方式

#### 串联电路
- **特点**：电流路径唯一，各元件依次连接
- **性质**：一个元件损坏，整个电路断开
- **应用**：开关串联在火线上（控制电路）

#### 并联电路
- **特点**：电流路径多条，各元件并列连接
- **性质**：一个元件损坏，其他元件仍可工作
- **应用**：家庭电路中的用电器并联（互不影响）

### 家庭电路
- **三孔插座**：左零右火上地（L-N-E）
- **开关位置**：串联在火线上（安全考虑）
- **用电器连接**：并联（互不影响）

### 磁场和磁感线
- **地磁场方向**：从地理南极指向地理北极
- **磁感线方向**：
  - 外部：从N极到S极
  - 内部：从S极到N极（形成闭合回路）
- **磁悬浮原理**：同名磁极相互排斥

### 右手螺旋定则（安培定则）
- **用途**：判断通电螺线管的磁极方向
- **方法**：右手握住螺线管，四指指向电流方向，拇指指向N极
